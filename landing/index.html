Manual Installation Guide

Recallify isn't on the Chrome Web Store yet. Here's how to install it in 3 easy steps:

1

Unzip the file

Download the .zip file above and extract it to any folder on your computer.

2

Open Extensions page

In Chrome, navigate to chrome://extensions or go to Menu → More Tools → Extensions.

3

Load the extension

Enable Developer Mode (top right toggle), then click Load unpacked and select the unzipped folder.
